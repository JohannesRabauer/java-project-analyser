= Context and Scope

== Business Context

The Java Project Analyser is a headless Model Context Protocol (MCP) server.
It sits between MCP-capable AI agents and the LLMs and database that execute analysis and search.
The server abstracts prompt engineering, AST parsing, vector indexing, and semantic retrieval behind a standard MCP tool interface.

The system supports two operating modes:

. **Analyse** — a multi-phase pipeline extracts structural and semantic knowledge from a Java codebase (mounted volume or Git clone) and indexes it into PostgreSQL with pgvector.
. **Query** — the client agent invokes MCP tools for pre-computed facts (instant) or on-demand LLM reasoning (10–30 s) over the indexed knowledge graph.

There is no browser UI.
All interaction happens through MCP clients (Claude Code, Cursor, VS Code Copilot) over stdio or SSE transport.

== External Systems

[cols="1,2,3", options="header"]
|===
| System | Direction | Responsibility

| MCP Clients (Claude Code, Cursor)
| Inbound
| AI agents that invoke MCP tools to trigger analysis and query results.
Connect via stdio (JSON-RPC 2.0) or SSE (HTTP streaming).

| PostgreSQL + pgvector
| Internal (owned by this deployment)
| Primary database.
Stores analysis job metadata, phase results, RAG chunks with vector embeddings, and query results.
The pgvector extension enables cosine similarity search over stored embeddings.

| Ollama
| Outbound (local profile)
| Runs open-weight LLMs (Llama 3, nomic-embed-text, etc.) on the host machine.
Receives prompts and embedding requests; returns structured text and float vectors.

| OpenAI API
| Outbound (openai profile)
| Hosted GPT-4o / text-embedding-3-small or equivalent.
Receives prompts and embedding requests; returns structured text and float vectors.

| Git Repository
| Outbound (optional)
| Remote source repository.
Cloned to a temporary directory when the user provides a URL instead of a pre-mounted volume.

| Source Project Volume
| Inbound (read-only)
| The Java codebase under analysis.
Mounted at `/workspace` inside the application container.
|===

== Technical Context

[cols="2,1,3", options="header"]
|===
| Channel | Protocol | Notes

| MCP Client ↔ Server (stdio)
| JSON-RPC 2.0 over stdin/stdout
| Primary transport for Claude Code.
Agent launches the server process directly; messages are newline-delimited JSON.

| MCP Client ↔ Server (SSE)
| HTTP on port 8080
| Secondary transport for Cursor and other HTTP-based clients.
Server-Sent Events stream for server-to-client notifications; HTTP POST for client-to-server requests.

| Server → PostgreSQL
| JDBC (port 5432)
| All persistent state: jobs, phase results, RAG chunks, query results.
pgvector `<=>` operator for cosine similarity queries.

| Server → Ollama
| HTTP on port 11434
| REST API for chat completions and embedding generation.

| Server → OpenAI
| HTTPS on port 443
| OpenAI Chat Completions API and Embeddings API.

| Server → Git Repository
| Git over HTTPS or SSH
| Delegated to `git clone`; credentials from the host environment.

| Server → Source Project Volume
| POSIX filesystem
| Read-only bind mount; no writes to the analysed project.
|===
