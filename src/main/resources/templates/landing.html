🔌 Connect Your AI Agent

SSE Endpoint: http://localhost:8080/mcp/sse

Claude Code

Add to your ~/.claude/mcp.json (or project-level .mcp.json):

{
  "mcpServers": {
    "java-analyser": {
      "url": "http://localhost:8080/mcp/sse"
    }
  }
}

Cursor

Add to .cursor/mcp.json in your project:

{
  "mcpServers": {
    "java-analyser": {
      "url": "http://localhost:8080/mcp/sse"
    }
  }
}

Any MCP Client (SSE)

Connect to http://localhost:8080/mcp/sse — standard MCP SSE transport.
